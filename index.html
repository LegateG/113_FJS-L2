JavaScript Lab 2

This page demonstrates JavaScript concepts like control flow, loops, and scope.

To see the results of the script, please open your browser's developer console.

Instructions to open the console:

  • Chrome/Edge/Firefox: Right-click anywhere on this page and select "Inspect" or "Inspect Element", then navigate to the "Console" tab.
  • Safari: Go to Safari > Preferences > Advanced, and check "Show Develop menu in menu bar". Then, from the Develop menu, select "Show JavaScript Console".
